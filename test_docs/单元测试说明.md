# 单元测试说明

## 后端测试

后端测试代码位于：

```text
fund_research_backend/src/test/java
```

运行方式：

```powershell
cd fund_research_backend
mvn test
```

覆盖率报告建议使用 JaCoCo 或 IntelliJ IDEA Coverage 生成，并将截图上传至：

```text
test_reports/backend
```

## 前端测试

前端测试代码位于：

```text
fund_research_front/src
fund_research_front/src/__tests__
```

测试文件命名：

```text
*.test.js
*.test.ts
*.spec.js
*.spec.ts
```

运行方式：

```powershell
cd fund_research_front
npm run test
```

覆盖率报告建议使用 Vitest Coverage 生成，并将截图上传至：

```text
test_reports/frontend
```

## 提交要求

每位成员至少提交一次与本人负责内容相关的测试代码、测试报告或测试说明，提交信息需要清晰说明修改内容。
